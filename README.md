## Git Konfigurasi
Tambahkan nama pengguna kamu di forum & email yang kamu gunakan ya!
<pre>git init
git config --global user.name "nama kamu di forum"
git config --global user.mail "email@yang.kamu.pakai"
</pre>

<pre>git commit -m "diskripsi"</pre>
